## OA办公管理系统

慕课网自学SSM框架的一个项目，使用Maven构建，采用MVC设计模式，项目分为三层设计，持久层整合了MyBatis，业务层配置了声明式事务，表现层实现了拦截器对未登录用户实现拦截。
 项目功能：
●  部门管理模块，员工管理模块的增删改查，其中员工管理涉及到和部门的多表联查。
● 个人用户中心，修改密码，退出等。
● 报销单模块，审核报销单，对报销单进行打款，不同的员工对报销单进行不同的操作 。
